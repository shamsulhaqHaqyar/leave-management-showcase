## Overview

Built for Lapis Group to replace paper-based leave tracking with a complete digital solution featuring automated approval workflows, real-time balance tracking, and role-based access control.

## Features

**Employee Portal**
- Submit leave requests (full-day or half-day)
- Track balance across 7 leave types
- View request history and status
- Email notifications

**Manager Dashboard**
- Approve/reject team requests
- Team analytics and statistics
- Leave balance tracker for direct reports
- Search and filter employees

**HR Dashboard**
- Organization-wide leave visibility
- Complete employee leave tracker
- User and balance management
- Export-ready reporting

**Workflows**
- Multi-step approvals (Manager → HR for sick leave > 1 day)
- Automatic balance deduction
- Color-coded status tracking
- Email notifications at each step

## Tech Stack

**Backend:** PHP 8.1, MySQL 5.7, PDO  
**Frontend:** HTML5, CSS3, Vanilla JavaScript  
**Security:** Bcrypt hashing, prepared statements, session-based auth  
**Hosting:** Hostinger (Production)
